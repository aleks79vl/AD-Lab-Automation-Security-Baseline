# AD-Lab-Automation-Security-Baseline v1.0.0

## Overview

Initial stable release of an enterprise-style Active Directory automation and security baseline toolkit.

The project generates PowerShell commands for Active Directory deployment, security hardening, rollback operations, compliance reporting, and deployment auditing.

---

## Features

### Active Directory Automation

- Organizational Unit creation
- User creation
- Group creation
- Computer creation
- Group membership automation

### Security Baseline

- Password Policy
- Workstation Security Policy
- Server Security Policy
- Microsoft Defender Policy
- Account Lockout Policy
- Audit Policy
- PowerShell Logging Policy

### Deployment Safety

- Configuration validation
- Rollback command generation
- Deployment logging
- Deployment reporting

### Compliance

- Security Compliance Report generation
